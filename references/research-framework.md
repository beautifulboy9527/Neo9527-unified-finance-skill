# 8阶段投研框架

> 整合自 stock-research-executor 的专业投研框架

## 概览

| 阶段 | 名称 | 核心问题 |
|------|------|---------|
| 1 | Business Foundation | 公司是做什么的？ |
| 2 | Industry Analysis | 行业处在什么周期？ |
| 3 | Business Breakdown | 公司怎么赚钱？ |

## Phase 1: Business Foundation (公司事实底座)

**Goal**: 建立对业务的事实性理解

### 研究内容

1. **核心业务和产品线**
   - 主要产品/服务是什么？
   - 收入占比如何？
   - 产品差异化程度？

2. **营收和利润构成**
   - 总营收和净利润
   - 过去5年 CAGR
   - 盈利能力趋势

3. **客户基础和应用场景**
   - 前5大客户占比
   - 客户集中度风险
   - 应用场景多样性

4. **在产业链中的位置**
   - 上游供应商议价能力
   - 下游客户议价能力
   - 替代品威胁

5. **近期战略变化**
   - 新产品/市场
   - 收购并购
   - 战略转型

### 输出格式

```
## Phase 1: Business Foundation

### 公司概述
[2-3句话描述公司主营业务]

### 营收构成
| 产品/服务 | 营收占比 | 毛利率 |
|----------|---------|--------|
| XXX      | XX%     | XX%   |

### 关键发现
- [关键发现1]
- [关键发现2]
- [关键发现3]
```

## Phase 2: Industry Analysis (行业周期分析)

**Goal**: 理解行业 dynamics 和竞争格局

### 研究内容

1. **行业周期阶段**
   - 复苏 / 扩张 / 衰退 / 收缩
   - 当前处于周期的哪个位置？

2. **供需 dynamics**
   - 行业总产能
   - 需求驱动因素
   - 供需平衡趋势

3. **价格机制**
   - 历史价格波动性
   - 定价权归属
   - 成本传导能力

4. **竞争格局 (CR5)**
   - 前5大厂商市场份额
   - 市场集中度趋势
   - 竞争激烈程度

5. **政策和外部变量**
   - 监管政策影响
   - 宏观经济影响
   - 技术和创新冲击

### 行业成熟度判断

| 成熟度 | 特征 | 投资逻辑 |
|--------|------|---------|
| 导入期 | 高增长、高不确定性 | 主题投资、选股 |
| 成长期 | 增长加速、竞争加剧 | 选择龙头 |
| 成熟期 | 增长放缓、行业整合 | 价值投资、格局优化 |
| 衰退期 | 负增长、退出 | 回避或转行 |

## Phase 3: Business Breakdown (业务拆解)

**Goal**: 理解公司如何赚钱

### 研究内容

1. **一句话业务本质**
   - 用一句话描述商业模式
   - 生意的本质是什麼？

2. **业务分部门量化**
   - 各业务板块收入和利润
   - 增长贡献分析
   - 盈利能力差异

3. **利润引擎**
   - 核心盈利来源
   - 边际利润变化驱动因素
   - 运营杠杆分析

4. **定价能力**
   - 产品差异化程度
   - 品牌溢价能力
   - 客户粘性

5. **子公司和非经常性项目**
   - 重要子公司及持股比例
   - 非经常性损益明细
   - 资产处置计划

## Phase 4: Financial Quality (财务质量)

**Goal**: 评估财务健康和盈利质量

### 研究内容

1. **关键指标趋势 (5-10年)**
   | 指标 | 2019 | 2020 | 2021 | 2022 | 2023 |
   |------|------|------|------|------|------|
   | 营收增速 | | | | | |
   | 净利润增速 | | | | | |
   | ROE | | | | | |
   | 毛利率 | | | | | |
   | 净利率 | | | | | |

2. **现金流交叉验证**
   - 经营现金流 / 净利润 > 1?
   - 自由现金流 / 净利润 > 0.8?
   - 现金流趋势是否稳定？

3. **异常筛查**

   | 异常类型 | 检测规则 | 危险信号 |
   |---------|---------|---------|
   | 应收账款 | 增速 > 营收增速 × 1.5 | 赊销激增，可能虚增收入 |
   | 存货 | 增速 > 营收增速 × 2 | 滞销积压 |
   | 毛利率 | 波动 > 行业均值 × 2 | 财务操纵嫌疑 |
   | 关联交易 | 占比 > 30% | 利益输送嫌疑 |
   | 非经常性损益 | 占比 > 30% | 主业盈利能力存疑 |

4. **财务风险识别**
   - 资产负债率 > 70%
   - 流动比率 < 1
   - 利息保障倍数 < 3

## Phase 5: Governance Analysis (股权与治理)

**Goal**: 评估管理层质量和资本配置

### 研究内容

1. **所有权结构**
   - 控股股东性质（国企/民企/外资）
   - 股权集中度
   - 实际控制人背景

2. **股份 overhang**
   - 限售股解禁计划
   - 股份回购情况
   - 二次发行计划

3. **管理层薪酬和激励**
   - 薪酬水平是否合理？
   - 股权激励覆盖范围
   - 高管是否与股东利益一致？

4. **资本配置记录 (ROIC)**
   - 过去5年 ROIC 趋势
   - 并购带来的回报
   - 分红和回购的恰当性

### 治理评分

| 评分 | 描述 |
|------|------|
| 5分 | 优秀的治理结构，股东利益高度一致 |
| 4分 | 良好的治理，有小的改进空间 |
| 3分 | 中等治理，存在一些结构性问题 |
| 2分 | 较差治理，中小股东权益受损 |
| 1分 | 极差治理，高风险信号 |

## Phase 6: Market Sentiment (市场分歧)

**Goal**: 理解牛市和熊市观点

### 研究内容

1. **牛市逻辑**
   - 主要看涨论点（3-5个）
   - 关键催化剂
   - 潜在上行空间

2. **熊市逻辑**
   - 主要看跌论点（3-5个）
   - 主要风险因素
   - 潜在下行空间

3. **关键辩论点**
   - 市场分歧最大的地方
   - 需要什么数据来验证？
   - 哪个观点更可能正确？

4. **关键验证节点**
   - 下次财报发布日期
   - 行业政策窗口
   - 宏观数据发布时间

## 强制交叉验证

1. **利润 vs 现金流**
   - 计算 经营现金流/净利润 3-5年
   - 如果比率 < 0.8，需要警惕

2. **公司 vs 同行**
   - 比较关键比率（利润率、增长率、估值）
   - 识别异常值并解释原因

3. **牛市 vs 熊市**
   - 确认两种观点都有实质性论据
   - 不要只看到想看的
