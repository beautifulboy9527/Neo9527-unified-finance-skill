### 数据时效性

| 数据类型 | 更新频率 | 延迟 |
|---------|---------|------|
| 美股行情 | 实时 | 15 分钟 (免费) |
| A 股行情 | 实时 | 基本实时 |
| 港股行情 | 实时 | 15 分钟 |
| 历史数据 | 每日 | T+1 |
| 财报数据 | 季度 | T+1~7 天 |

### 数据稳定性问题

**已知问题**:
1. AkShare 偶尔连接失败 (网络问题)
2. Alpha Vantage demo key 限流严重
3. 东方财富 API 无官方保障

**建议**:
- 增加更多备用数据源
- 配置正式 API key
