### 6. 一键报告 ⚠️

**问题**: Windows 终端 emoji 编码错误
```
UnicodeEncodeError: 'gbk' codec can't encode character '\U0001f680'
```

**原因**: stock-market-pro 的 rich 库使用 emoji，Windows GBK 编码不支持

**解决方案**:
1. 修改 yf.py 移除 emoji 或改用 UTF-8 输出
2. 或在调用前设置 `PYTHONUTF8=1` 环境变量
