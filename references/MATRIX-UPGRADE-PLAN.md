# 金融 Skills 矩阵 - 打磨升级计划

**日期**: 2026-04-07  
**战略定位**: 专精分工 + 数据冗余 + 链式流程

---

## 一、整体架构理念

### 核心原则

1. **专精分工**: 每个 skill 专注一个领域，做到极致
2. **数据冗余**: 重叠数据源作为备用，确保可靠性和准确性
3. **链式流程**: 后期支持自定义 skills 链式调用
4. **统一入口**: unified-finance-skill 作为默认入口，stock-assistant 兼容旧用法

### Skills 矩阵

```
┌─────────────────────────────────────────────────────────────┐
│                    统一入口层                                │
│  unified-finance-skill (主)  │  stock-assistant (兼容)       │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    数据源层 (多源冗余)                        │
│  ┌─────────────┬─────────────┬─────────────┬─────────────┐ │
│  │ yfinance    │ agent-stock │ akshare     │ (备用)      │ │
│  │ (美股)      │ (A/H 股)    │ (全市场)    │             │ │
│  └─────────────┴─────────────┴─────────────┴─────────────┘ │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    功能层 (专精分工)                          │
│  ┌──────────┬──────────┬──────────┬──────────┬──────────┐  │
│  │ 图表生成 │ 相关性   │ 流动性   │ 情绪分析 │ 警报     │  │
│  │ (market) │ (corr)   │ (liq)    │ (sent)   │ (alert)  │  │
│  └──────────┴──────────┴──────────┴──────────┴──────────┘  │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                    链式流程层 (后期)                          │
│  示例：行情 → 图表 → 评分 → 警报 → 报告                      │
└─────────────────────────────────────────────────────────────┘
```

## 三、链式流程设计 (后期)

### 示例流程 1: 日常监控

```
行情查询 → 技术信号解读 → 警报检查 → 生成日报
   ↓           ↓              ↓          ↓
quote.py   analyze.py    alert.py   daily_report.py
```

### 示例流程 2: 深度分析

```
行情查询 → 图表生成 → 相关性分析 → 流动性分析 → 情绪分析 → 完整报告
   ↓          ↓            ↓            ↓            ↓          ↓
quote.py   chart.py   correlation.py liquidity.py  sentiment.py report.py
```

### 示例流程 3: 选股流程

```
条件筛选 → 基本面分析 → 技术面分析 → 情绪分析 → 加入 Watchlist
   ↓           ↓            ↓            ↓            ↓
screener.py  fundamental.py technical.py  sentiment.py  watchlist.py
```
